Classe permettant de manipuler des requètes SQL dyanmiquement en simplifiant leur écriture.

A utiliser pour les très grosses requètes afin de préserver leur cohérence en limitant les erreurs de développement et utilisateur.

author Romain Laï-King
version 0.3
